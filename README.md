# Dotfiles

Personal configuration files for macOS development environment.

## Structure

- **aerospace/** - AeroSpace tiling window manager configuration
- **nvim/** - Neovim configuration with Lua setup
- **zsh** - Shell configuration (Oh My Zsh + Powerlevel10k)
- **tmux/** - Terminal multiplexer configuration

## Key Features

### AeroSpace
- Tiling window management with vim-like keybindings
- Automatic workspace assignment for apps (ChatGPT, Music, Telegram, Figma, Browser)
- Custom gaps and layout preferences

### Neovim
- Lua-based configuration
- LSP support with Mason
- Git integration (gitsigns)
- AI (CodeCompanion)
- File navigation (Telescope, Oil)

### Zsh
- Oh My Zsh framework
- Powerlevel10k theme
- Custom aliases (eza, git shortcuts)

## Dependencies

- [Oh My Zsh](https://ohmyz.sh/)
- [Neovim](https://neovim.io/)
- [AeroSpace](https://github.com/nikitabobko/AeroSpace)
- [Tmux](https://github.com/tmux/tmux)
